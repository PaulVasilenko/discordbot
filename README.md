# Panda Bot

## Commands
 - `!pandabot` - Ответит !pandabot

### Confify
 - `!confify [imageurl]` - Натягивает лица на картинку поверх чужих лиц.

### Highlighter

#### Game codes
Все команды требуют геймкод. Пока что поддерживается два геймкода
 - lol (League of Legends)
 - ow (Overwatch)

#### Commands

 - `!subscribegame | !subg` - Подписаться на хайлайты по игре
 - `!unsubscribegame | !unsubg` - Отписаться от хайлайтов по игре
 - `!startsession | !starts` - Начать сессию, автоматически хайлайтит всех, кто подписался на хайлайты
 - `!stopsession | !stops` - Прекратить сессию
 - `!joingame | !jg` - Присоединиться к игре
 - `!leavegame | !lg` - Покинуть игру
 - `!currentplayers | !cp` - Показывает список играющих, без хайлайта
 - `!highlightforlobby | !hfl` - Хайлайтит всех играющих
 - `!highlightnotinlobby | !hfnl` - Хайлайтит всех, кто подписался но не в игре
